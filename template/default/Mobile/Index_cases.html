爱北欧风 享受慢生活

设计师：朱家桢

设计师：陈国建

设计师：周叶靖宇

设计师说：

干净的北欧风

简洁的浅白

搭配明亮的黄

以简及多的空间概念

整体感舒适自然

整个客厅是清爽的质感

素净的白

明媚的黄

安静的椰子棕

优雅的豆绿

将空间的气质提升到新的高度

心中向往的北欧

不可或缺的是一抹米色阳光。

平顶搭配吊灯

再加上四五个小射灯

空间的明度便呈现出来

工作闲暇之余

躺在慵懒的棉麻质沙发上

窗边斜阳余晖透过纱幔撒在书本和主人的脸庞

是呀     日子的情趣也不过如此

黑胡桃木制的圆形茶几

暖白的灯光照在茶几上

将北欧的温柔感融为一体

简洁又不失现代感
